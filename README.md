# 1. 데이터명
언어 학습 애플리케이션 '듀오링고(Duolingo)'의 구글 플레이스토어(Google Playstore) 리뷰 데이터셋

# 2. 출처
https://play.google.com/store/apps/details?id=com.duolingo

# 3. 데이터셋
파이썬에 내장된 구글 드라이브 스크래퍼를 활용하여 어플 '듀오링고'의 리뷰를 크롤링하였다.

## 3-1. 데이터 구축 규모
- 전처리 전: 50000건
- 전처리 후: 4500건

# 4. 전처리 과정 소개
## 4-1. 리뷰 데이터 50000건 수집
'듀오링고'의 구글 플레이스토어 리뷰를 최신 순으로 50000건을 수집하였다.

## 4-2. 한글이 포함된 데이터셋만을 추출
외국어로만 구성된 것이 아닌 한국어가 무조건 포함이 되어있는 데이터셋만을 추출하였다. (50000건 -> 31620건)

## 4-3. 결측치 제거
'reviewCreatedVersion', 'appVersion'열에서 결측치가 발견되어 결측치를 제거하였다. (31620건 -> 29869건)

## 4-4. 정제 과정
- 이모지 제거
- 숫자 + 단위 처리
- 불용어 처리
- 용언(동사/형용사) 처리
- 숫자 + 단위 다시 합치기
  
## 4-5. 별점별 리뷰 수 조정
별점 중 2점의 개수가 912건으로 제일 적었으며, 각각 랜덤으로 1점부터 5점까지 900건을 추출하였다.

## 4-6. 필요한 열 선정
id, 별점, 리뷰 내용, 정제가 끝난 리뷰 데이터만을 열로써 추출
